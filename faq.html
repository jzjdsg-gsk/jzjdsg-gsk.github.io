Pi Network 是一個新興的加密貨幣平台，致力於讓更多人參與到加密貨幣的生態系統中來。通過手機應用程式，使用者可以免費獲得 Pi 幣。

在 Pi Network app 中註冊並登入後

點擊首頁 圖示即可開始

點擊週期為24小時，輕鬆就能獲得 Pi 幣。

Pi 幣在2/20 正式上架主網

OKX 行情數據顯示

PI 幣 20 日下午 4 點以 1.51 美元開盤。

Pi 幣當前價格可至首頁上方查看。
KYC 驗證是Pi Network 為了確保用戶的真實性，並且防止假帳戶作弊等行為的身分驗證機制，正式上架主網後，如果想將Pi 幣成功提領到各交易所，就必須要完成Pi幣KYC 身分驗證，而且需要等待審核。
